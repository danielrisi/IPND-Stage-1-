Lesson 1: Basics of the Web and HTML

The World Wide Web

The World Wide Web was created in the 1990s. It's basically a huge collection of HTML or HyperText Markup Language files which are linked amongst themselves and other files (images,video,etc.) via hyperlinks.

Components of the Web

  • Browser - interprets files found on the web
  • Internet - worldwide computer network
  • HTTP - Hypertext Transfer Protocol
  • Servers - computers that host the files on the web

Basic HTML Elements

HTML elements are used to format and add funcionalities to web pages. Opening tags are written between < > and are closing tags with </>. Examples:

  • <b> bold
  • <em> - italic
  • <br> - line break
  • <p> - paragraph
  • <div> - "box" container element
  • <span> - "inline" container element

Some elements require attributes in their opening tag:

  • <a href="url"> - used to create hyperlinks by adding a URL attribute called "href"
  • <img scr="url"> - displays an image from source attribute "src"

Lesson 2: Creating a Structured Document

Developer Tools

Developer Tools is a Chrome feature that allows developer to inspect and experiment with a web page without altering the source code.

HTML-CSS-DOM

  • HTML - HyperText Markup Language - the standard markup language used to create web pages.
  • CSS - Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language.
  • DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes, Grids and Rules

It's very helpful to structure a web page by thinking of boxes, so that any element of the page can be repositioned on a grid.
